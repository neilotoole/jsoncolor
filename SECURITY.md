# Security Policy

## Supported Versions

Only the latest minor release line receives security fixes.

| Version  | Supported          |
| -------- | ------------------ |
| v0.9.x   | :white_check_mark: |
| < v0.9.0 | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** — do not open a public
issue for them.

- **Preferred:** use GitHub's private vulnerability reporting via the
  repository's **Security** tab → **"Report a vulnerability"**
  ([advisories/new](https://github.com/neilotoole/jsoncolor/security/advisories/new)),
  if enabled.
- **Otherwise:** email the maintainer at **neilotoole@apache.org**.

Please include reproduction steps and the affected version. You can expect an
acknowledgement, and we will coordinate a fix and a disclosure timeline with you.
